# USANA 家用产品管理系统 (Pro)

这是一个专为 USANA 产品设计的家用库存管理 PWA 应用。

## 🚀 主要功能
- **库存清单**：实时追踪澳洲与中国区产品的库存量。
- **智能预警**：低于设定的阈值时自动提醒补货。
- **多币种支持**：自动根据实时汇率计算库存的总估值（AUD/CNY）。
- **交易记录**：详细记录每一次入库（采购）和出库（自用/借出/售出）。
- **数据安全**：所有数据均保存在本地浏览器，支持 JSON 文件的导出备份与导入。

## 🛠 开发技术
- **React 19**
- **TypeScript**
- **Vite** (构建工具)
- **Tailwind CSS** (样式)
- **Lucide React** (图标)
- **Recharts** (统计图表)

## 📦 部署说明
该项目已适配 **Netlify** 自动构建：
- **Build command**: `npm run build`
- **Publish directory**: `dist`

## 📱 安装方式
1. 在手机浏览器打开部署后的 URL。
2. 点击浏览器的分享/菜单按钮。
3. 选择 **“添加到主屏幕”** 即可作为独立 App 使用。
